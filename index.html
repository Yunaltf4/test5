À propos de moi

Hey! Je m'appelle Tony et je suis étudiant au Collège Maisonneuve. Je suis dans le programme Technique d'intégration Multimédia. En ce moment j'étudie surtout les langages Javascript, CSS et Html pour environ trois mois. Ces trois langues sont importantes pour le développement de site web et application moderne. J'ai hâte de continuer à les apprendre et améliorer mes connaissances et compétence.

Comment j'ai fait ce Projet

Recherche D'idées

Dans ce projet je devais me donner un thème pour un site web pour un jeu vidéo fictive. J'ai décidé de faire un site pour un jeu qui se jouerait dans l'espace. J'ai, donc, mit des vaisseaux spatiaux, l'intérieur de ces vaisseaux et des vaisseaux d'ennemies qu'on pourrait croiser dans le jeu. Avec ce Mood Board que j'ai fait, je l'ai passer dans Adobe Color pour me donner des couleurs principales. Et par la suite, j'ai trouvé une police qui irait bien avec le thème.

Appliquation des Idées

Pour l'application de mes idées j'ai utilisé beaucoup de couleurs qu'Adobe Color m'a donné pour faire mon site. Dans la barre de navigation ou le hearder/nav on peut voir plusieurs menus qui peuvent nous rendre dans d'autre partie du site, en plus d'un bouton pour télécharger le jeu. Juste en dessous on peut voir le titre du jeu « Space invasion », un « À propos du jeu » et des images qui nous montrent à quoi va ressembler le jeu.
